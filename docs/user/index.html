Wprowadzenie

Projekt udostępnia zarówno często wykorzystywane oraz nietrywialne funkcjonalności pomagające w pisaniu paczki. W szczególności jest on rozwinięciem dawnego oi.h. Spis treści:

Korzystanie z biblioteczki

Pobieranie

Aby korzystać z biblioteczki należy sklonować repozytorium z gitlaba. W template-package znajduje się skrypt klonujący jedynie niezbędne rzeczy.

Aktualizacje

Aby aktualizować biblioteczkę wystarczy wpisać komendę: git fetch.

Include

W najprostszym wariancie aby skorzystać z większości funkcjonalności wystarczy napisać #include "staszic-library/oi.hpp"
